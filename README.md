## Steps to use the app

- Use the command "git clone https://github.com/pankajvasnanitech123/users-management-app.git" to clone the application.
- Then, use the command "composer install" to install the packages for the application.
- Change database credentials before the migration process.
- Then, use the command "php artisan migrate" to migrate the tables.
- Then, use the command "php artisan db:seed" to seed the testing data required to login into the application.
